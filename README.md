# 🤖 AI HR Assistant – Resume Screening Automation (n8n)


## 📌 Overview
This project is an AI-powered HR assistant designed to automate the resume screening process. It leverages workflow automation using **n8n** to evaluate candidate resumes, generate a score, and assist HR teams in making faster and more consistent hiring decisions.

The system reduces manual effort by integrating AI-based analysis with automated email workflows for both HR and candidates.

---

## 🚀 Features

- 📄 **Resume Analysis**
  - Automatically evaluates candidate resumes using AI
  - Assigns a score out of 10 based on predefined criteria

- 📊 **Candidate Scoring System**
  - Standardized scoring for fair evaluation
  - Helps HR quickly shortlist candidates

- 📧 **HR Notification**
  - Sends candidate score and evaluation summary to HR via email
  - Enables HR to approve or reject candidates

- 🔄 **Automated Candidate Communication**
  - Sends acceptance/rejection emails based on HR decision
  - Ensures timely communication with candidates

- ⚙️ **Workflow Automation (n8n)**
  - Fully automated pipeline from resume input to final decision
  - Easy to extend and customize

---

## 🛠️ Tech Stack

- **n8n** – Workflow automation
- **OpenAI API** – Resume evaluation & scoring
- **Email Integration (Gmail)** – Notifications


---

## 🔁 Workflow

1. Candidate submits resume
2. AI analyzes the resume and generates a score (out of 10)
3. Score and evaluation are sent to HR via email
4. HR reviews and takes action (Approve/Reject)
5. Based on HR decision:
   - ✅ Approval → Candidate receives selection email
   - ❌ Rejection → Candidate receives rejection email

- Standardizing candidate evaluation processes
